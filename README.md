# 🏍️ Geo-Moto Matrix

A simple and fast-paced 2D bike racing game built with Python and Pygame. This project was developed as a final lab submission for the Computer Graphics Lab (CSE-328) at Feni University. Instead of using image files, all graphics are dynamically rendered using mathematical shapes.

## 🌟 Features
* **Dual Controls:** Play with Mouse (Swipe left/right) or Keyboard (Arrow keys).
* **Power-ups:** Collect Coins (Yellow diamond) for a time bonus, and Shields (Purple diamond) for 5 seconds of invincibility.
* **High Score System:** Automatically saves your best survival time using File I/O (`bike_highscore.txt`).
* **Dynamic Speed:** The game gets faster and harder the longer you survive!
